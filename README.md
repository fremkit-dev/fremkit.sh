# fremkit.sh

Content for `https://fremkit.sh`, deployed via Cloudflare Pages.

- `/` rewrites to `/install.sh` with a `200` response for curl-friendly installs.
- `/index.html` is the human-readable landing page.
- `install.sh` is a published copy.
- Source of truth for `install.sh`: `fremkit-dev/fremkit:install.sh`.
